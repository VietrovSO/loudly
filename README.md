# Loudly
Результатом проектування БД є створення системи, яка здатна надавати інформаційні послуги з видачі необхідних даних та їх оновлення. База даних містить інформацію про пісні, їх назву, автора і забезпечує можливість пошуку та відбору даних. У відповідності з предметною областю система будується з урахуванням таких особливостей:
- кожна пісня представлена у межах свого жанру;
- для кожної пісні можна отримати її назву та відомість про автора;
- для кожного альбому можна отримати його назву, дату виходу та детальний опис;
- за жанром, обраним користувачем, формується список пісень;
